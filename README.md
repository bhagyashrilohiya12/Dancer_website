# Dancer/Choreographer Portfolio Website

This is a modern React portfolio website template for a dancer or choreographer, built with Vite for speed and easy updates.

## Features
- Home, Bio, Gallery, Events, and Contact sections
- Easy to update content in simple files
- Responsive, modern design

## Getting Started

1. **Install dependencies:**
	```sh
	npm install
	```
2. **Start the development server:**
	```sh
	npm run dev
	```
3. **Open your browser:**
	Visit the local address shown in the terminal (usually http://localhost:5173)

## Editing Content
- Update your bio, gallery images, events, and contact info in the `src/content/` folder (to be created).
- Replace placeholder images in `src/assets/` with your own.

## Build for Production
```sh
npm run build
```
